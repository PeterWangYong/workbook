# cursor

cursor用于调整鼠标指针样式



## resize

```css
div {
  cursor: n-resize;
  cursor: e-resize;
  cursor: s-resize; 
  cursor: w-resize;
  cursor: ne-resize;
  cursor: nw-resize;
  cursor: se-resize;
  cursor: sw-resize;
  cursor: ew-resize;
  cursor: ns-resize;
  cursor: nesw-resize;
  cursor: nwse-resize;
}
```
