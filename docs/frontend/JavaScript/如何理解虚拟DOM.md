> 若无说明，DOM指真实DOM

虚拟DOM可以理解为JS和DOM之间的缓存，避免JS直接操作DOM。

## 要解决的问题

1. 通过数据绑定实现视图的自动更新，即数据驱动，降低代码的复杂度。

2. 通过计算更新前后的差异，再将差异应用到DOM上，减少了不必要的DOM操作，提高了性能

   > 操作DOM是昂贵的，很耗性能
