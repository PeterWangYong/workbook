# 语言

## 短路原则

逻辑运算符遵循短路原则，只要确定符号前面的真假，即可确定返回值

1、只要“||”前面为false，无论“||”后面是true还是false，结果都返回“||”后面的值。

2、只要“||”前面为true，无论“||”后面是true还是false，结果都返回“||”前面的值。

3、只要“&&”前面是false，无论“&&”后面是true还是false，结果都将返“&&”前面的值;

4、只要“&&”前面是true，无论“&&”后面是true还是false，结果都将返“&&”后面的值;

```js
console.log(1 && 2) // 2
console.log(0 && 1) // 0
console.log(1 || 2) // 1
console.log(0 || 2) // 2
```
